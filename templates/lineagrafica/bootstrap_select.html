Bootstrap Select

Bootstrap-select requiere jQuery v1.9.1+, el componente dropdown.js de Bootstrap y CSS de Bootstrap. Si aún no usa Bootstrap en su proyecto, puede descargar aquí una versión precompilada de los requisitos mínimos de Bootstrap v3.4.1. Si usa bootstrap-select con Bootstrap v4+, también necesitará Popper.js. Para conocer todos los requisitos de Bootstrap v4, consulte Primeros pasos. Una versión precompilada de los requisitos estará disponible en una próxima versión de bootstrap-select.

Bootstrap Select Documentation

Uso

CSS

Copie y pegue la hoja de estilo <link> en su <head> para cargar el CSS.

<link rel="stylesheet" href="../node_modules/bootstrap-select/dist/css/bootstrap-select.min.css">

Js

Copie y pegue el siguiente <script> y Function cerca del final de sus páginas.

<script src="../node_modules/bootstrap-select/dist/js/bootstrap-select.min.js"></script>
