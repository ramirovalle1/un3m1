Tagify

Transforma un campo de entrada o un área de texto en un componente de etiquetas, de una manera fácil y personalizable, con un gran rendimiento y una huella de código pequeña, ampliada con funciones.

Tagify Documentation

Uso

CSS

Copie y pegue la hoja de estilo <link> en su <head> para cargar el CSS.

<link rel="stylesheet" href="/static/bootstrap5/libs/@yaireo/tagify/dist/tagify.css">

Js

Copie y pegue el siguiente <script> y Function cerca del final de sus páginas.

<script src="/static/bootstrap5/libs/@yaireo/tagify/dist/tagify.min.js"></script>
