Quill Documentation

Uso

CSS

Copie y pegue la hoja de estilo <link> en su <head> para cargar el CSS.

<link rel="stylesheet" href="/static/bootstrap5/libs/quill/dist/quill.snow.css">

Js

Copie y pegue el siguiente <script> y Function cerca del final de sus páginas.

<script src="/static/bootstrap5/libs/quill/dist/quill.min.js"></script>
                                  
                                      
                                         if ($("#editor").length) new Quill("#editor", {

                                            theme: "snow"
                                        });
